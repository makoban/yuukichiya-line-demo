# 本番LIFF化で必要になるLINE側情報

今回の静的デモを見るだけなら不要です。

本番または実際のLINEメニューに接続する段階では、以下が必要です。

- LINE公式アカウントの管理権限
- LINE DevelopersのProvider / Channel
- LIFFアプリのEndpoint URL
- LIFF ID
- リッチメニュー各ボタンの遷移先URL
- Messaging APIで設定する場合はチャネルアクセストークン
